# iPhoneセットアップ手順（タニメモ）

実際のURL・APIトークン入りの完成版は `ios-shortcut-setup.local.md`（git管理外）を参照。以下はテンプレート。

## 1. ホーム画面に追加

1. Safariで `https://tanimemo.<アカウント名>.workers.dev` を開く
2. 設定画面でAPIトークンを貼り付けて保存
3. 共有ボタン → 「ホーム画面に追加」→ 名前「タニメモ」

## 2. 共有ショートカット（テキスト・URL用）

ショートカットAppで新規ショートカットを作成する。

1. 名前:「タニメモに送る」
2. 情報(i) → 「共有シートに表示」をオン
3. 受け取る内容: 「URL」「テキスト」だけをオンにする
4. アクション「URLの内容を取得」を追加:
   - URL: `https://tanimemo.<アカウント名>.workers.dev/api/share`
   - 方法: POST
   - ヘッダ: キー `Authorization` / 値 `Bearer <APIトークン>`
   - 本文を要求: フォーム
   - フォームフィールド: キー `text` / 種類 テキスト / 値 「ショートカットの入力」

## 3. 共有ショートカット（画像用）

同様にもう1つ作成する。

1. 名前:「タニメモに画像を送る」
2. 「共有シートに表示」をオン、受け取る内容: 「イメージ」のみ
3. アクション「URLの内容を取得」:
   - URL・方法・ヘッダは上と同じ
   - 本文を要求: フォーム
   - フォームフィールド: キー `file` / 種類 ファイル / 値 「ショートカットの入力」

## 4. 使い方

- SafariでURLを共有 → 共有シート → 「タニメモに送る」
- 写真を共有 → 共有シート → 「タニメモに画像を送る」
- 送った内容はアプリの「受信」タグに入る（次回同期時に反映）
- 複数枚の画像を一度に共有すると、1枚ずつ別メモになる（2026-07-21 実機確認済み）
